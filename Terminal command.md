cd "<path>"           : change directory.
			Note: cd .. to go back to parent folder.

dir                   : show all elements in current folder (Common in Windows/CMD)

ls [flags]            : list directory contents. In Git Bash, ls is usually colorful by default.
			Note: Use ls --color to force colors

clear	              : clear the console screen.

mkdir "<folder name>" : create a new folder

touch "<filename>"    : create a new file

echo "text" > |>> "<filename>" : 
					> : overwrite
					>>: insert/append

cat "<file_name>"      : show content inside a file

diff <file_1> <file_2> : compare two files to see the differences

rm [-flag] <target>    : remove a file
			-r: remove folder (recursive)
			-d: remove empty folder
